# MAPQUE

[![HitCount](http://hits.dwyl.com/waterbobs/MAPQUE.svg)](http://hits.dwyl.com/waterbobs/MAPQUE)
 
 - 2번째 개인 프로젝트
 - google play store : 심사중


# 특징
 
 1. 카카오 맵을 이용한 주소 검색
 2. 내 위치, 검색한 위치 두가지 위치 확인 가능
 3. 검색한 위치는 검색 목록 화면에 저장됨
 4. 위치 카카오톡에 공유 가능
